
# Mon site fait avec Nuxt et symfony 5.4

juste pour fun, j'ai décidé de tester une librairie qui s'appelle locomotive-scroll avec nuxt version 2.

## Installation

### Frontend

Après avoir cloner le code :

```bash
  npm install
```
Pour le lancement : 

```bash
    npm run dev
```
